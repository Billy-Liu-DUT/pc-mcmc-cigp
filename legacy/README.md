# Legacy Research Scripts

This directory contains the original exploratory scripts from the research
prototype. They are preserved for reference while the public package API lives
under:

```text
pc_mcmc_cigp/
```

Recommended current entry points:

- `examples/discover_hbr_mechanism.py`
- `examples/optimize_epoxidation_cigp.py`
- `experiments/fig3_hbr_discovery.py`
- `experiments/fig4_epoxidation_bo.py`
- `scripts/plot_fig3.py`
- `scripts/plot_fig4.py`

The legacy scripts may still run, but they use older global configuration files
and are not the preferred interface for new work.
