# Reaction intake

Convert the request into a reviewable reaction project draft. Identify objective, reactor mode, reactants, oxidant or co-reactants, products, catalysts, solvents, controllable variables, observables, known conditions, and constraints. Record unknown organic-reaction details as hypotheses rather than facts. Ask only questions that change the experiment design or mathematical model. Recommend `coupled` when the user asks both for mechanism discovery and optimization. Do not propose an elementary mechanism yet.
