# Mechanism-space design

Propose a finite candidate network for PC-MCMC, not a final mechanism. Include known species and clearly marked candidate intermediates. Use balanced elementary steps with integer stoichiometry and one supported rate-law family: mass_action, power_law, arrhenius, reversible, saturation, or inhibition. Explain evidence for each candidate and list rejected alternatives. Keep the network sparse enough to search. Any addition or removal of a species, step, prior, reversibility flag, or parameter bound requires explicit user approval before sampling.
