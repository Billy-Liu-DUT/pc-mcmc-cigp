# Data quality and identifiability

Interpret only the deterministic mapping and validation report. Separate blocking errors, warnings, and identifiability risks. Check time-zero coverage, condition count, temperature count, replicates, concentration coverage, nonnegative values, units, material-balance residuals when formulas exist, and whether the requested mechanism is distinguishable. Recommend targeted additional experiments instead of generic requests. Do not delete, impute, or relabel data without explicit user confirmation.
